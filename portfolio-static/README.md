# Static portfolio

This is a plain HTML, CSS, and JavaScript portfolio. It has no dependencies, no build step, and no server.

## Preview locally

Open `index.html` in any browser, or use a static-file preview extension in your editor.

## Publish on GitHub Pages

1. Copy the contents of this folder to the repository root **or** publish this folder through your preferred Pages workflow.
2. In GitHub repository **Settings → Pages**, choose **Deploy from a branch**.
3. Select the branch and `/ (root)` folder containing `index.html`.

GitHub Pages will host it directly—no `pnpm`, Node.js, or build command is required.
